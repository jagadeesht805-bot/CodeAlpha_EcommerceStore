# 🛒 CodeAlpha E-Commerce Store

## 📌 Project Overview

This is a Full Stack E-Commerce Store developed as part of the CodeAlpha Full Stack Development Internship.

The application allows users to browse products, register, log in, add products to the cart, and place orders.

---

## 🚀 Features

- User Registration
- User Login
- Product Listing
- Product Details
- Search Products
- Shopping Cart
- Checkout
- Order History
- Responsive Design

---

## 🛠️ Technologies Used

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## ⚙️ Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
